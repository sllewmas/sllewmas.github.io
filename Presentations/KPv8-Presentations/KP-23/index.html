Enharmonic Spellings

Enharmonic: Notes that have the same pitch, but are spelled differently

A♭ = G♯

Keys can be enharmonic as well.

F♯ Major = G♭ Major

Enharmonic Spellings

Enharmonic spellings are used by composers for a variety of reasons.

  1. To better show the desired voice-leading.
    • Generally, every note wants to resolve by a minor second whenever possible.
    • ♭'s want to resolve down.
    • ♯'s want to resolve down.
  2. To make things easier for the performer.

Enharmonic Reinterpretation

The possible harmonies that can be reinterpreted are:

  1. Mm7 chord (V7)
  2. diminished seventh (°7)
  3. augmented triad (+) [RARE]
  4. French augmented sixth chord (Fr+6) [RARE]

Enharmonic Modulations

Enharmonic Modulations: a type of modulation in which the common chord is reinterpreted enharmonically to fit into the second key.

The actual spelling of the chord is not important—it might be spelled as it would appear in the first key, or in the second key.

What is important is that the common chord can be heard as a sensible chord in both keys.

Using the Mm7 Sonority

Remember: we can reinterpret any Mm7 chord as a Ger+6 chord.

Any V7 chord or secondary V7 in the first key can be reinterpreted as a Ger+6 chord in the new key.

The reverse is also possible — a Ger+6 in the first key can become a V7 or secondary V7 in the second key.
